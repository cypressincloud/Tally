# Tally - 用户须知

## 📖 项目简介
Tally 是一个纯本地的记账项目，拥有自动记账、照片备份、自动续费提醒、多币种、关联资产等核心功能。

## ⚙️ 功能默认情况
- **记账助手设置**：“开启屏幕自动记账”关闭，“开启资产功能”开启
- **自动资产设置**：“开启自动资产”关闭
- **分类预设管理**：“开启二级分类”关闭
- **记账界面默认显示**：“结余”
- **货币单位设置**：“开启货币单位”关闭，默认“￥”
- **照片备份设置**：“开启照片备份”关闭。备份为纯本地存储（原理是复制一份至指定路径），推荐新建以“.”开头（如 `.Tally`）的空白文件夹作为存储路径
- **主题设置**：“跟随系统”
- **极简模式**：关闭

> 💡 **隐藏彩蛋**：长按应用内“用户须知”界面左上角的标题，可以取消激活高级设置。

## 🛡️ 项目权限清单
- **自启动**：用于自动记账进程的后台保活。
- **无障碍**：实现屏幕读取与自动记账的核心权限。
- **悬浮窗**：用于 1 像素保活机制及记账时的即时弹窗交互。
- **通知**：提供实时状态反馈，自动记账功能稳定后可手动关闭。

## 🤖 自动记账配置
使用“无障碍管理器”锁死 Tally 的无障碍权限。微信高版本对无障碍有限制，解决方案是下载“一木记账”，同时锁死 Tally 和一木记账的无障碍权限。

*注：“无障碍管理器”和“shizuku”在123网盘链接里面的“辅助软件”中可以找到，具体使用方法请观看B站相关视频。*

## 🔒 隐私安全承诺
由于使用到的权限较为敏感，**Tally 承诺完全不具备联网功能，未来也不会添加。**
如有云同步等需求，建议参考开源代码自行扩展。欢迎关注酷安 “Tally” 项目，我们将不定时发布版本更新。
